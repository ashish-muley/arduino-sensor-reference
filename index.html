Microcontrollers

Start with a board you have

Specs, pinouts, IDE setup, and a tested Blink LED sketch for each board — including the quirks (ESP8266's active-LOW LED, ESP32's board-dependent LED pin).

Arduino Uno

ATmega328P · 16 MHz · 5V

The default starting board for most projects.

Arduino Nano

ATmega328P · 16 MHz · 5V

Same chip as the Uno, breadboard-friendly.

Arduino Mega 2560

ATmega2560 · 16 MHz · 5V

Many I/O pins and extra serial ports.

ESP32

Xtensa LX6 · 240 MHz · 3.3V

Dual-core with Wi-Fi + Bluetooth.

ESP8266 / NodeMCU

L106 · 80 MHz · 3.3V

Budget Wi-Fi board for IoT.

STM32

ARM Cortex-M · 3.3V

Coming soon — ARM development boards.
Communication

Understand how sensors actually talk

The difference between copying a sketch and understanding it. Each protocol explains its wiring convention, key terms, and which sensors in this repo use it.

UART

TX · RX · baud

Serial Monitor, GPS, Bluetooth modules.

I²C

SDA · SCL · address

OLED, MPU6050, BMP280 — shared 2-wire bus.

SPI

MOSI · MISO · SCK · CS

RFID, SD cards, displays — fast 4-wire bus.

CAN

CAN-H · CAN-L

Automotive & industrial networking.
