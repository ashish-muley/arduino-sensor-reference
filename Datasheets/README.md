# Datasheets

This is an **index**, not a duplicate file store. Each sensor's datasheet link lives in its own folder README under [`../Sensors/`](../Sensors/) and [`../Component-Reference/`](../Component-Reference/README.md), right next to its code — so nothing is separated from the component it documents.

This page lists where to get datasheets straight from the manufacturer, plus a quick index of the documented components.

## Always Use Official Sources

Avoid random PDF aggregator sites — they're frequently outdated or watermarked. Go to the manufacturer directly:

| Manufacturer | For | Link |
|---|---|---|
| Arduino | Official board documentation | [docs.arduino.cc](https://docs.arduino.cc/) |
| Microchip | ATmega328P, ATmega2560 | [microchip.com](https://www.microchip.com/) |
| Espressif | ESP32, ESP8266 | [espressif.com](https://www.espressif.com/en/support/documents/technical-documents) |
| STMicroelectronics | STM32, VL53L1X | [st.com](https://www.st.com/) |
| Texas Instruments | LM35, DRV5053, sensors | [ti.com](https://www.ti.com/) |
| Bosch Sensortec | BMP280, BME680, BNO055 | [bosch-sensortec.com](https://www.bosch-sensortec.com/) |
| Analog Devices | AD8232, MAX30102, ADXL345 | [analog.com](https://www.analog.com/) |
| Sensirion | SHT31, SCD41 | [sensirion.com](https://sensirion.com/) |
| NXP | MFRC522 (RC522) | [nxp.com](https://www.nxp.com/) |
