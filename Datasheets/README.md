# Datasheets

This is an **index**, not a duplicate file store. Each sensor's actual datasheet PDF lives inside its own folder under [`../Sensors/`](../Sensors/) (e.g. `Sensors/Distance/HC-SR04/Datasheet.pdf`), right next to its code and wiring diagram — that way nothing is separated from the component it documents.

This page just lists where to get datasheets straight from the source, and links to where each one lives in this repo once added.

## Always Use Official Sources

Avoid random PDF aggregator sites — they're frequently outdated, watermarked, or missing revision notes. Go to the manufacturer directly:

| Manufacturer | For | Link |
|---|---|---|
| Arduino | Official board documentation | [docs.arduino.cc](https://docs.arduino.cc/) |
| Microchip | ATmega328P, ATmega2560 datasheets | [microchip.com](https://www.microchip.com/) |
| Espressif | ESP32, ESP8266 technical reference manuals | [espressif.com/en/support/documents](https://www.espressif.com/en/support/documents/technical-documents) |
| STMicroelectronics | STM32 datasheets | [st.com](https://www.st.com/) |
| Texas Instruments | LM35 and other TI sensor datasheets | [ti.com](https://www.ti.com/) |
