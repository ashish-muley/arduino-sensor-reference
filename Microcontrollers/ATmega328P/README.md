# ATmega328P Family

Classic 8-bit AVR boards — the most common starting point for learning embedded basics. Uno, Nano, and Mega 2560 are grouped here because they share the same programming model and most code is interchangeable between them (Mega uses the larger ATmega2560, same architecture family).

| Board | Best For |
|---|---|
| [Arduino Uno](Arduino-Uno/) | Default choice for learning and most single-sensor projects |
| [Arduino Nano](Arduino-Nano/) | Same capability as Uno, smaller footprint for breadboards |
| [Arduino Mega](Arduino-Mega/) | Projects needing many I/O pins, multiple sensors at once, or extra serial ports |

Each board folder contains its own specs, pinout, IDE setup steps, and a tested Blink LED sketch.
